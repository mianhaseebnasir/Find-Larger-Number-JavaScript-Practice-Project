
<h1>🔢 Find Larger Number – JavaScript Practice Project :</h1>

<h1>📄 Description :</h1>


A beginner-friendly JavaScript project that compares two numbers and displays the larger value using DOM manipulation and conditional logic.


<h1>📌 Project Overview :</h1>


Find Larger Number is a simple interactive web application built using HTML and JavaScript.
The application allows users to enter two numbers and determines which number is larger — or if both numbers are equal.

This project is part of fundamental JavaScript practice, focusing on:

• DOM Manipulation

• User Input Handling

• Conditional Statements

• Basic Form Handling


<h1>🚀 Features :</h1>


• User-friendly number input fields

• Form submission without page reload

• Comparison logic using JavaScript

• Displays result dynamically in a readonly input field

• Uses conditional statements (if, else if, else)

• Console logging for debugging practice


<h1>🛠️ Technologies Used :</h1>


• HTML5 – Structure of the web page

• JavaScript (ES6) – Logic & interactivity

• DOM Methods (getElementById)

• Template Literals


<h1>📂 Project Structure :</h1>


5th-Practice/
│── index.html

The entire application logic is written inside the <script> tag within the HTML file.


<h1>⚙️ How It Works :</h1>


1. User enters two numbers.

2. On clicking the Check button:

• The getNumber() function is triggered.

• Values are fetched using document.getElementById().value.

• Numbers are converted using Number().

3.Conditional statements compare both numbers.

4.The result is displayed inside a readonly input field.


<h1>💡 Learning Concepts :</h1>


This project helps in understanding:

• Type Conversion in JavaScript (String(), Number())

• Form submission handling with onsubmit

• Preventing page reload using return false

• Dynamic value update in input fields

• Writing clean and readable JavaScript logic
